HTML br tag

The ' br ' tag in HTML. The browser does not recognize new lines and paragraph formatting in the text. If we want to start a new line, we need to insert a line break with the help of the ' br ' tag. It is a self-closing tag and does not require any type of closing tag. It is used for addresses, poetry, or in situations where line breaks are needed.
